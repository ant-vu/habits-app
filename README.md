## **Habit Tracker**

## **Installation**

To install Habit Tracker, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/ant-vu/habits-app.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the project:

```bash
npm run dev
```
